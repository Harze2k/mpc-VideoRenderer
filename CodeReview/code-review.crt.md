## Code Review

---

### Quick Start
   
#### Add Snippet To Existing Review

- Select the code you want to add
- Right-click and choose `Add To Code Review`
- Select the target review file
- Snippet will be inserted with:
    - Syntax highlighting
    - GitHub repository link
    - Direct IDE file link

#### Copy A Shareable Snippet

- Select the code snippet
- Right-click and select `Copy Snippet To Clipboard`
- Markdown snippet with syntax highlighting is now in your clipboard
- Paste it anywhere - includes GitHub and IDE links

#### Tips

- Recently created reviews can be quickly reopened from `Tools > Code Review Tools > Recent Code Reviews`
- Supports multiple programming languages
- GitHub links are available for both public and private repositories
- Jetbrains links require [Jetbrains Toolbox](https://www.jetbrains.com/toolbox-app/) installed - supports IntelliJ IDEA, WebStorm, PyCharm, AppCode, CLion, PhpStorm, RubyMine, Rider, GoLand, RustRover
